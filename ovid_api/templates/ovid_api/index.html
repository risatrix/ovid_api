An API for Ovid
Ovid's love poetry in a simplified, code-friendly structure

About this Site

This site serves as a place to grab Ovid's poetry with code. Using it assumes some knowledge of API's, but I'll try to lay things out for people who haven't done that before. If you are familiar with APIs, you might want to look directly at the docs page.

Why Do I Care?

If you need material for a bot project, or a Javascript project, or whatever else, it's here.

How do I use this thing?

To get a list of all works, try: /authors/ovid/works/

The author and work use an all lower case slug, and everything else is numbers.

To get a poem, you need to use a book number and poem number, even if that's not how you'd traditionally cite the poem.

So, the same way you'd use: /authors/ovid/works/amores/books/1/

You need to add a book number: /authors/ovid/works/heroides/books/1/

To get a particular line: /authors/ovid/works/ars/books/2/lines/34/ If you already know what you want you can use the short version, remembering to include the book and poem number: /ovid/medicamina/1/1/34

Technical details

To get the results in JSON for your code, you need to add =json to the end of the url.
